Exemples

Ce site regroupe les exemples que nous avons utilisés dans le livre HTML5 et CSS3. Chaque chapitre qui contient des exemples, dispose de son propre dossier, d'une page index.html ainsi que d'un fichier style.css si le chapitre traite de CSS3.

Chaque page inclut une feuille de style stylebase.css pour les besoins de mise en page de ce site. Cette page d'accueil est un autre exemple de ce qu'on peut faire en CSS3 (Sélecteurs et Animations), n'hésitez donc pas à en regarder le code source !
